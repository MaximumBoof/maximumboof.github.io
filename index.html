Agent Based Model

Model details

Using python, an agent-based model has been created. This model is animated and sees a user defined number of agents move around an imported environment, where they are able to interact with each other and the environment.

Due to the web data scraping parts of the code, expect the model to take roughly 30 seconds for the figure window to open.

Model features

  • A graphical user interface with spinboxes and buttons
  • User defined number of agents, number of iterations and neighbourhood size
  • Agents are able to interact with the environment and other agents
  • Animation of the model working
  • Feature's web data scraping for agent starting locations
  • A environment imported from a CSV file
  • The model is able to be restarted with different variables without having to reopen the window
